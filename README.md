# Stock Market Analysis

This project analyzes stock price data using R to evaluate trends, returns, and volatility.

## Overview
- Compared AAPL, TSLA, and SPY
- Calculated daily returns
- Measured volatility using standard deviation
- Visualized performance using normalized price and return plots

## Key Insight
Tesla showed the highest returns but also the greatest volatility, demonstrating the trade-off between risk and return.

## Tools Used
- R (quantmod, tidyverse, ggplot2)
- Quarto

## Files
- `stock-analysis.qmd` – source code and analysis
- `stock-analysis.html` – rendered report
